# Конные программы: как ищем и обновляем

Этот проект хранит и показывает программы для ребенка/подростка с опытом верховой езды:
- с жильем, обучением, оплатой, компенсацией дороги, либо бесплатным участием;
- по приоритетным странам без Азии (Шенген, Ирландия, UK, USA, Канада, Австралия, НЗ, Грузия, ЮАР, Аргентина).

## Что именно ищем

Подходят только варианты, где есть хотя бы одно условие:
- дают жилье;
- оплачивают обучение;
- есть оплата участия/работы/съемки;
- компенсируют дорогу;
- полностью бесплатная программа.

Отбрасываем:
- только для взрослых без подростковых треков;
- без horse-riding requirement;
- без прозрачных условий;
- сомнительные схемы без официальных страниц.

## Базовый промпт для поиска

Скопируй в любую нейросеть:

```text
Ищи по всему миру opportunities для ребёнка/подростка с опытом верховой езды.

Регионы: Schengen countries, Ireland, UK, USA, Canada, Australia, New Zealand,
Georgia, South Africa, Argentina. Азию не включай.

Показывай только варианты, где есть хотя бы одно:
- оплачивают поездку;
- дают жильё;
- оплачивают обучение;
- бесплатная программа;
- оплачиваемый кастинг/съёмка/проект.

Источники: equestrian sites, casting platforms, job boards, summer camps,
internships, exchange/volunteering programs.

Для каждого результата верни:
title, country, age range, type, pay, travel support, accommodation, training,
deadline, application link, short reason why it fits.

Если что-то неочевидно — пометь needs manual check.
Отбрасывай варианты без подтвержденного horse-riding requirement.
```

## Кнопки/режимы работы поиска

## 1) Начать поиск
- Запускай ручной цикл поиска.
- Результат: список новых программ + список изменившихся программ.

## 2) Периодический поиск
- Рекомендуемо: 1 раз в неделю.
- Проверяем:
  - новые программы;
  - изменения дедлайнов;
  - закрытие набора.

## 3) Отчет по изменениям
- Формат отчета:
  - `NEW`: новые программы;
  - `UPDATED`: изменились условия/дедлайн;
  - `CLOSED`: набор закрыт или дедлайн прошел.

## Как обновлять файлы проекта

## Источники данных
- Все сырье/отчеты складываются в `research/`.
- Витрина (публичная страница): `programs.html`.
- Canvas в Cursor: `equestrian-programs.canvas.tsx`.

## Добавление новой программы
1. Добавь объект в `PROGRAMS` (в `programs.html` и в Canvas-файле).
2. Заполни поля:
   - `name`, `org`, `category`, `country`, `baseCountry`;
   - `age`, `minAge`, `deadline`, `tone`;
   - `support`, `pay`, `desc/description`, `url`;
   - `check/needsCheck` (если нужно);
   - `closed` (если набор закрыт).
3. Проверь фильтры:
   - категория, страна, возраст, архив.

## Перевод программы в закрытые (Архив)

Если программа больше не актуальна:
- поставь `closed: true` у этой программы;
- обнови `deadline` (например, "Закрыто: 15.08.2026");
- добавь комментарий в `check/needsCheck` с причиной закрытия.

Поведение интерфейса:
- закрытые не видны в основных фильтрах;
- закрытые видны в фильтре `Архив`.

## Мини-чеклист перед публикацией
- все ссылки открываются;
- дедлайны читаются однозначно;
- возраст (`age` + `minAge`) корректен;
- закрытые программы помечены `closed: true`.
