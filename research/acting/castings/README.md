# Кастинги — актёрское мастерство (16–18+)

Папка для **периодической проверки** актёрских кастингов (film/TV/theater) для подростков 16–18+.

| Файл | Назначение |
|------|------------|
| [`castings-registry.json`](castings-registry.json) | Главная база: платформы, агентства, active/watch/archive |
| [`MONITORING_DATABASE.md`](MONITORING_DATABASE.md) | Сводка для человека |
| [`CHECKLIST_проверка.md`](CHECKLIST_проверка.md) | Промпт для Cursor |
| [`Israel_анализ_2026-06-12.md`](Israel_анализ_2026-06-12.md) | Глубокий разбор рынка IL |

**На сайте:** категория **«Кастинги в актёрском мастерстве»** (ids **1021–1045**; Israel **1041–1045**).

**Автоархив после проверки:** `node scripts/archive-expired-castings.mjs` (`--dry-run` — предпросмотр).

**Счётчик при «Открытые» (июнь 2026):** см. [`MONITORING_DATABASE.md`](MONITORING_DATABASE.md).

Конные кастинги — отдельно: [`research/equestrian/castings/`](../equestrian/castings/).
